01

Project Overview

This project involved producing a complete set of residential building drawings from scratch using AutoCAD. The assignment required adherence to standard drafting conventions, including proper layering, annotation styles, dimension standards, and title block formatting.

The deliverable consists of two separate drawing sheets: an Architectural Plan showing the structural layout, room arrangement, furniture, dimensions, and construction details; and an Electrical Plan showing circuit routing, and outlet/fixture placement overlaid on the floor plan. Together, they form a coordinated drawing package representing a complete residential design.

Drafting Discipline: Both sheets were drafted using AutoCAD's model space / paper space workflow, with separate layers for annotations, dimensions, electrical symbols, and construction geometry enabling clean, print-ready outputs at specified scales.
// ARCHITECTURAL

Floor Plan & Layout

Detailed floor plan with wall thicknesses, door and window schedules, room labels, and fully dimensioned geometry following standard residential conventions.

// ELECTRICAL

Electrical Systems Plan

Overlay drawing depicting circuit runs, outlet locations, switch layouts, lighting fixtures, and electrical panel information per NEC conventions.

// STANDARDS

Proper title blocks, revision history, graphic scale, standard line weights, and annotation styles conforming to industry drafting practice.

Drafting Standards Applied

📏

Line Weight Hierarchy

Object lines, hidden lines, centerlines, and dimension lines drawn with correct lineweight differentiation per ANSI/AIA conventions.

🏷️

Layer Management

Separate AutoCAD layers for walls, dimensions, annotations, electrical, and title block, enabling independent visibility and printing control.

🔠

Text & Annotation Standards

Consistent font styles, text heights, and note formatting throughout both drawing sheets for a professional, readable output.

📐

Scale & Viewport Control

Paper space viewports set to standard architectural scales with properly scaled annotations and dimensions in each viewport.

⚡

NEC Electrical Symbology

All electrical symbols drawn per National Electrical Code (NEC) standards with a complete legend included on the electrical sheet.

📋

Title Block & Sheet Format

Standard title block includes project title, scale, date, sheet designation, revision history, and engineer/drafter signature block.
